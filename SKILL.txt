# Portfolio Agustín Skill

Esta skill ayuda a construir, corregir y mejorar el portfolio profesional de Agustín.

El objetivo es crear una página moderna, profesional, limpia, responsive y con identidad propia. El portfolio debe presentar a Agustín como estudiante de Ingeniería en Sistemas y futuro desarrollador backend/fullstack, con interés en arquitectura de software, automatización, cloud e IA aplicada a sistemas reales.

---

## Objetivo principal

El portfolio debe funcionar como centro profesional de Agustín.

Debe mostrar:

- Perfil profesional.
- Presentación personal.
- Stack tecnológico.
- Proyectos.
- CV descargable.
- GitHub.
- LinkedIn.
- Contacto.
- Casos de estudio.
- Evolución futura hacia proyectos grandes.

La página no debe parecer una plantilla genérica. Debe verse profesional, seria, moderna y preparada para mostrar proyectos reales.

---

## Stack del proyecto

Asumir que el proyecto usa:

- React.
- TypeScript.
- Vite.
- CSS común, CSS Modules o Tailwind, según lo que ya exista en el proyecto.
- Arquitectura basada en componentes.

No agregar librerías nuevas salvo que el usuario lo pida explícitamente.

---
